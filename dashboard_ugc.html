ネガ投稿はクリエイティブの問題ではなく、CS・予約UXへの改善インプットとして別ラインで活用。「満席・予約しづらい」は裏返せば需要超過のシグナルでもある。

4軸タグとの接続 — UGCを「次のクリエイティブ」のアイデア源に 統合価値の核

UGC投稿にも広告と同じ4軸（訴求タイプ / セグメント / モチーフ / デザインパターン）タグを自動付与。UGCでは多いのに広告では薄いモチーフ＝生活者が自発的に語っているのに広告が追いついていない領域＝次のクリエイティブ候補。

実装にあたっての要件確認事項

以下4点は推測で埋めず、先方確認が必要（要先方確認）:
  • X API の契約状況・費用負担 — 投稿検索には有料プラン（Basic以上）が必要。現契約の有無・プラン・費用負担の整理。
  • Instagram Graph API の権限 — ビジネスアカウント連携の有無、メンション・ハッシュタグ取得が可能な権限範囲。
  • 収集対象の範囲 — 自社メンションのみか、指定ハッシュタグを含むか、リプライ・引用まで含むか。
  • 収集頻度 — リアルタイム / 日次バッチ / 週次。API費用と鮮度のトレードオフをどこに置くか。
